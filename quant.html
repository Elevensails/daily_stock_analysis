🔌 MCP 对接方案

Vibe-Trading 支持 MCP（Model Context Protocol）服务器模式。对接后，WorkBuddy 专家可以在对话中直接调用量化能力：

// ~/.workbuddy/mcp.json 添加 { "mcpServers": { "vibetrading": { "command": "python", "args": ["-m", "vibetrading.mcp"], "env": { "OPENAI_API_KEY": "sk-...", "LONGBRIDGE_TOKEN": "..." } } } }

对接后在 WorkBuddy 对话中可直接问：

  • vibe research "招商银行 30 天红利低波回测"
  • vibe bench --strategies dip_buy,ma_cross --asset 159915 --start 2018-01-01
  • vibe event "百合花 半年报业绩窗口 事件驱动分析"
  • vibe shadow --capital 100000（影子账户无风险实盘跟踪）

📋 对接步骤

  • ① 安装 Vibe-Trading：pip install vibetrading 或 docker pull hkuds/vibe-trading
  • ② 配置 MCP：在 ~/.workbuddy/mcp.json 添加 vibetrading 服务器配置
  • ③ WorkBuddy 授权：在 WorkBuddy UI → 连接器 → 信任 vibetrading
  • ④ 对话调用：在 WorkBuddy 专家对话中输入 vibe research/bench/event/shadow 命令
  • ⑤ 报告同步：Vibe-Trading 报告生成后手动推送到 gh-pages，与本页综合展示
